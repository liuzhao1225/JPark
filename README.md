# JPark 建平乐园

欢迎来到建平乐园！这里是一个 A 股分析工作台，每天跟踪市场动态、挖掘个股机会。我们结合实时行情数据、历史走势分析和市场热点资讯，力求做出有理有据的投资判断。

## 数据来源

- **大致数据**：使用 [mcp-cn-a-stock](https://github.com/elsejj/mcp-cn-a-stock) MCP 服务，提供股票基本信息、行情、财务、技术指标
- **细致数据**：使用 [baostock](http://baostock.com/) 编写 Python 脚本获取详细历史数据

## 分析规范

- 每次分析必须调取**最近五个交易日的日内走势数据**，判断短期趋势和量价关系
- 使用 baostock 获取近5日分钟级 K 线数据
- **必须联网查询近期市场热点**：搜索该股票或所属板块的近期新闻、政策动向，纳入分析报告的"市场热点与消息面"章节
- **所有画图使用 Mermaid** 语法，不自定义颜色，保持默认主题
- **默认直接输出不写文件**：分析结果直接在对话中呈现，只有明确要求时才写入文件（一次调研往往需要多轮探索）

## 文件组织规范

所有内容按**北京时间当天日期**组织在对应文件夹中：

```
JPark/
├── 20260211/
│   ├── SH600000_浦发银行_走势分析.md
│   └── script/
│       └── fetch_sh600000.py
├── 20260212/
│   └── ...
└── README.md
```

- **日期文件夹**：格式 `YYYYMMDD`，如 `20260211`
- **Markdown 文件**：直接放在日期文件夹中，命名格式 `股票ID_名称_分析内容.md`
- **Python 脚本**：放在日期文件夹下的 `script/` 子文件夹中
